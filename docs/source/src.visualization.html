src.visualization.visualize module¶

src.visualization.visualize.crossvalidate_pipeline_scores(X, y, pipelines, n_splits, random_state)¶

crossvalidates all pipelines in the provided dictionary and returns scores (R2, neg-MAE, neg-MRE)

Parameters

  • X (dict) – Dataframe with the predictors

  • y (series) – Pandas series with the target values

  • pipelines (dict) – dictionary with the name of the model as key and pipeline as value

  • n_splits (int) – how many splits to do in crossvalidation

  • random_state (int) – random state for splitting

Returns

Dataframe with scores calculated for each fold and model

Return type

dataframe

src.visualization.visualize.plot_scores(scores, show_costs=False, save=False, plotname=None)¶

Generates BoxPlots for all metrics

Parameters

  • scores (dataframe) – Dataframe with columns model, metric, fold and value (output from crossvalidate_pipelines)

  • show_cost (boolean) – Plot the computation cost metrics

  • save – Save created plots to reports/figures/

src.visualization.visualize.train_and_plot_prediction_metrics(X_train, y_train, X_test, y_test, pipelines)¶

Trains the pipelines with train data, predict test data with trained models and the plots MAE, MSE and R2 metrics

  • X_train (dataframe) – Training data features

  • y_train (array) – Training data target

  • y_test (array) – Test data target

  • pipelines (dict) – dictionary with the name of the model as key and pipeline as value
